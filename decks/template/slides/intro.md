## Who is this for?
  - You use or have used git on a daily basis with other people on a project. <!-- .element: class="fragment" data-fragment-index="1" -->
  - You have exposure to strategies and workflows such as feature branching. <!-- .element: class="fragment" data-fragment-index="2" -->
  - You are interested in leveling up your git skill set. <!-- .element: class="fragment" data-fragment-index="3" -->
  - You have been using git for a while so you understand the basics of adding changes and commits. <!-- .element: class="fragment" data-fragment-index="4" -->
  - You are looking for relatively easy ways to speed up new feature integrations, avoid merge conflicts as much as possible, and sometimes perform code reviews.  <!-- .element: class="fragment" data-fragment-index="5" -->
