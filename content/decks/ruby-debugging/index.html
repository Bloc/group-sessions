pry

Install via:


            $ gem install pry
            $ pry

Then add:


            require 'pry'

into your program.

Setting a breakpoint 


def powersOfTwo(n)
  newArr = []
  (0..n).to_a.each_with_index do |x, i|
    binding.pry # breakpoint set
    newArr.push(2*i)
  end

  newArr
end
